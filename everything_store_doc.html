Everything Store

This is a full-stack e-commerce project I built using Laravel while learning the framework. It includes a storefront where users can browse products by category, add items to a cart, and place orders. I also created an admin panel to manage products, orders, and users. Through this project, I practiced working with Laravel’s MVC structure, handling databases, and building a complete web application from backend to frontend.

📌 Project Overview

What the project does:
The application simulates a real online shopping experience. Customers can explore products, filter them by category, and complete an order process. At the same time, the admin dashboard makes it easy to add, update, or delete products, and track incoming orders.

Who it is for:
This project is designed for small businesses or individuals who want a basic online store to showcase and sell their products. It can also serve as a starting point for developers who want to build more advanced e-commerce platforms.

What problem it solves:
It simplifies the process of selling products online by providing a structured system for managing products and handling customer orders. Instead of managing everything manually, the admin can control the store from one place, while customers get a smooth and organized shopping experience.

⚙️ Tech Stack

Backend: Laravel (PHP)

Frontend: HTML Tailwind JavaScript Blade

Authentication: Laravel Breeze

Database: MySQL

Libraries & Tools: Chart.js DomPDF Laravel Notifications

Other features: Eloquent ORM, Routing, Middleware, CRUD operations, MVC architecture, RESTful APIs

🚀 Features

  • Browse products with a clean and responsive storefront
  • Filter products by categories for easier navigation
  • View detailed product pages with full information
  • Add products to a shopping cart and manage quantities
  • Simple checkout flow to place orders
  • Admin dashboard to manage products (create, update, delete)
  • Order management system for tracking customer purchases
  • User authentication using Laravel Breeze
  • Analytics dashboard with charts using Chart.js
  • Real-time notifications for order updates using Laravel Notifications
  • PDF generation for orders or reports using DomPDF

🧠 What I Learned

While building this project, I gained a deeper understanding of Laravel and how to structure a full-stack application using the MVC pattern.

I improved my skills in working with databases using Eloquent ORM, and learned how to design relationships between different tables in a real project.

I also practiced building authentication systems using Laravel Breeze and handling user roles between customers and admin.

On the frontend side, I learned how to create responsive and clean interfaces using Blade, Tailwind CSS, and JavaScript.

In addition, I explored useful tools like Chart.js for displaying data visually and DomPDF for generating PDF files, which helped me understand how to extend a Laravel project with external libraries.

Overall, this project helped me think more like a developer by solving real problems and organizing a complete web application from scratch.
